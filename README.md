# Análisis Histórico de Tecnologías

Este proyecto permite analizar la popularidad y evolución histórica de lenguajes de programación y tecnologías, combinando datos de GitHub y Stack Overflow.

Incluye generación de datos reales o simulados, análisis estadístico (CAGR, volatilidad, tendencias) y visualizaciones interactivas con Plotly.
### Características principales

1. Obtiene datos históricos desde:

  Stack Overflow API — actividad por etiquetas (preguntas).
  
  GitHub API — repositorios creados por año y lenguaje.

2. Genera métricas avanzadas:

  CAGR (Crecimiento anual compuesto)
  
  Crecimiento reciente y total
  
  Ranking compuesto ponderado

3. Visualizaciones interactivas:

Comparación de crecimiento (CAGR vs Momentum)

Distribución de mercado anual

Ranking anual animado de tecnologías

4. Exporta un reporte Excel con todos los resultados.
